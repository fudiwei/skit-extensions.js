一个多功能的 JavaScript 内置对象原型链扩展方法工具库。

（[English Documentation](./README.md)）

---

## 特性

-   对部分内置对象扩展了若干常用原型链方法；

-   各模块相互独立，可单独拷贝到项目中使用；

-   支持 TypeScript。

---

## 用法

### 安装：

```shell
npm install @skit/extensions
```

### 导入：

```javascript
/* require */
require('@skit/extensions');

/* import */
import '@skit/extensions';
```

### API:

请参考 _/types/\*.d.ts_ 下的声明文件。
